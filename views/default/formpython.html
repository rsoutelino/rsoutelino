Curso de introdução ao Python como ferramenta para Oceanógrafos

Python é uma linguagem de programação interpretada que pode ser usada interativamente. Isto facilita o desenvolvimento de códigos científicos para manipular conjuntos de dados extensos e complexos programas computacionais. Adicionalmente, o Python por si só possui uma extensa gama de módulos para executar tarefas do dia a dia de um usuário de sistema operacional, que usualmente requerem o uso de múltiplas linguagens. A base de ferramentas voltadas à computacão científica, disponíveis gratuitamente, é notavelmente extensa. Módulos como Numpy, Scipy e Matplotlib possibilitam a execucão de tarefas complexas com matrizes de dados, com bibliotecas matemáticas para manipulacão e análise de dados. Python tem bibliotecas disponíveis para portar códigos C e Fortran em sua linguagem, o que torna ultra-rápidas tarefas que são conhecidamente mais lentas em linguagens interpretadas. Muitas ferramentas voltadas à Oceanografia e Ciências da Terra ja estão disponíveis, como 1) manipulacão de formatos como NetCDF, HDF, GRIB; 2) OpenDAP; 3) análise harmônica de maré; 4) cálculo de propriedades termodinâmicas da água do mar e 4) visualizacão gráfica/plotagem de dados totalmente customizável, com qualidade para publicacão. Python é ideal para lidar com pré- e pós-processamento associados a modelagem numérica.
O repositório git do curso contém o material didático relevante: GitHub - rsoutelino


Porque não MATLAB?

Texto retirado da homepage do pesquisador Rob Hetland.

MATLAB pode ser considerada atualmente a linguagem padrão para análise e plotagem em aplicacões oceanográficas, tanto para modelagem numérica como para oceanografia observacional. Tal fato se deve a razões históricas, e não só pelo MATLAB em si, mas pelo surgimento das primeiras toolboxes voltadas a Oceanografia e Ciências da Terra. Entretanto, muitas desvantagens e problemas associados ao uso do MATLAB podem ser citados, e são amplamente conhecidos pela comunidade. Lentidão, instabilidade, vazamentos de memória, bugs em geracão de figuras, servidores de licensa e código fechado são exemplos. É uma linguagem flexível, dinâmica, e uma vez que aprende-se a trabalhar neste arcabouço matricial, ela pode ser rápida e poderosa. Migrar para uma outra linguagem implica no aprendizado de um novo conjunto de técnicas de programação e geração de uma nova gama de toolboxes. Este investimento inicial certamente demanda tempo, portanto espera-se que haja importantes benefícios. Acreditamos que migrar para Python valha o esforço pelas seguintes razões:

  • Python é uma linguagem de programação extremamente poderosa e moderna. Isto significa que pode-se usar Python não apenas para análise e plotagem, mas também para as demais tarefas que a computação científica possibilita, como gerenciamento de arquivos e diretórios, automatização de tarefas, downloads e interatividade com web em geral, criação de interfaces gráficas.
  • Python é verdadeiramente orientada a objeto. Isto significa que os programas tendem a ficar mais fortes, claros, flexíveis e fáceis de desenvolver, ler e debugar, por causa da natureza da programação em Python. É muito simples converter/organizar um conjunto randômico de ferramentas em um pacote coeso para uma dada finalidade.
  • Existem muitos pacotes disponíveis gratuitamente que executam tarefas que comummente queremos fazer. Em particular, os pacotes Numpy e Matplotlib têm uma interface muito similar à do MATLAB, tornando a curva de aprendizado bem reduzida.

Algumas vantagens do Python em relação ao Matlab:

  • Python é uma linguagem de script completa e poderosa. Módulos científicos são escritos nesta filosofia. Por exemplo, a criação de classes é muito simples e funcões podem ser passadas como argumentos de classes ou outras funções, funcionalidade esta não disponível no MATLAB. Existe um número de tipos de variáveis bem maior, que são muito poderosos, como listas, dicionários e tuplas.
  • É fácil de portar códigos escritos em C e Fortran. Existem inúmeras ferramentas para isto e a maioria das ferramentas científicas já escritas em Python se aproveitam disso, então o código roda super rápido.
  • Python é open source, gratuito, portável, está em constante desenvolvimento e lançamento de novas versões ocorrem com uma frequência muito maior que no MATLAB. A comunidade de desenvolvedores é extensa e ativa. Python roda em todos os computadores modernos e arquiteruras, podendo ser inclusive executado em tablets e smartphones.


Estrutura do Curso


  • Aula 1: Conceitos de produtividade, reproducibilidade e trabalho colaborativo. Introdução das ferramentas de produtividade (editores de texto, python, git) e preparação dos computadores. Conceitos básicos de python: Listas, tuplas dicionários, strings, condicionais, loops, exceções. Programação abstrata. Trabalhando com arrays (matrizes). Criação e manipulação de arrays usando o Numpy. Indexação, fatiamento e operações com arrays. Manipulações básicas de dados, como interpolacões e filtros. Leitura de dados ascii. Exemplo prático de leitura e plotagem de dados de CTD.
  • Aula 2: Técnicas intermediárias de programação científica. Interação com o sistema, automatização de tarefas, interação com arquivos. Orientação à objeto, funções, classes, encapsulamento. Desenvolvimento de software: tornando o programa re-utilizável. Determinando propósitos do software, requerimentos. Trabalho colaborativo: contrubuindo coletivamente a uma mesma base de códigos, dividindo/delegando tarefas, comunicação eficiente, desenvolvimento simultâneo sem conflitos. Preparando um pacote para distrubuição: determinando inputs, funcionalidades e outputs, escrevendo documentação.

    Consiste em elaborar um pacote simples e útil para leitura, tratamento e visualização de dados de CTD. Este deve preencher requisitos mínimos de desenvolvimento e compartilhamento de software livre.

Tutoriais de instalação


Aqui forneceremos dicas para fazer uma instalação de python com bibliotecas para uso científico. Independentemente do sistema operacional, os softwares necessários são os seguintes:

  • Editor de texto para desenvolvimento de código. À gosto do usuário, mas sugerimos SublimeText2
  • Python 2.7
  • iPython mais recente
  • Módulos e bibliotecas Python: Scipy, Matplotlib, Basemap toolkit do Matplotlib, netCDF4, Seawater, Pandas, e Pydap


Instalação multiplataforma de todos os pacotes

Para os menos experientes que procuram uma instalação rápida, sem dores de cabeça, sugerimos o uso de uma distribuição para python científico chamada “Anaconda” (Anaconda). Esta tem instaladores automáticos para Windows, Linux e Mac, com todos os pacotes necessários.


Instalação em sistema operacional Debian/Ubuntu Linux via apt-get

Neste caso, todos os pacotes são baixados e compilados para o seu sistema em particular, ocupando menos espaço e com performance ligeiramente melhor. Segue as instruções abaixo:

  • sudo apt-get install build-essential gfortran nco libnetcdf-dev libhdf5-serial libhdf5-serial-dev libkernlib1-gfortran netcdf-bin hdf5-tools

  • Instalar o netcdf mais recente para sua arquitetura.

  • sudo apt-get install python-scipy python-matplotlib python-mpltoolkits.basemap ipython python-pip python-setuptools python-netcdf python-pandas

  • sudo easy_install seawater pydap netcdf4
